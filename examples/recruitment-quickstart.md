# Recruitment Quickstart

How to use Kolvera skills for a recruitment workflow — from mapping target companies to placing candidates.

## The recruitment workflow

```
Map the market → Screen candidates → Build RMPs → Launch campaigns → Place
```

## Step 1 — Map your target market

```
Map the market for [your niche]. Target [buyer titles] at [company types] with [size range] employees in [locations].
```

This runs the full `map-the-market` skill: ICP creation, Deep Research, job scraping, contact enrichment, hot list segmentation. Takes 30-60 minutes with approval gates.

## Step 2 — Screen and qualify candidates

After screening a candidate, update their record:

```
Move [candidate name] to active, grade them A, add these screening notes: [summary]. Set next action to "send to [company]" on [date].
```

## Step 3 — Take a candidate to market (RMP)

When a candidate passes the qualification gate:

```
Take [candidate name] to market. They are a [role title] looking for [perm/contract/both] at [salary/rate]. Create the RMP, cross-reference my ICP, and build the campaign.
```

This runs the full `rmp-campaign` skill: RMP creation, company map building, prospect enrichment, campaign construction, and activation.

## Step 4 — Monitor campaigns

Daily:
```
Check my campaigns
```

Weekly:
```
Weekly review
```

## Step 5 — Handle replies

When a prospect replies positively:
→ Prepare the candidate pack
→ Get written authority to represent (if not already obtained)
→ Submit within 24 hours

Update the prospect:
```
Move [prospect] to meeting_booked. Note: "[summary of reply]". Set next action to "send candidate profile" on [date].
```

## Tips

→ **Lead with candidates, not services.** Cold outreach with "do you need recruitment help" does not convert. Specific, available candidates do.
→ **One RMP per candidate.** Each RMP can feed multiple campaigns (by vertical or engagement type).
→ **Never name the candidate in emails.** Use "he/she" only until authority is confirmed and a profile is requested.
→ **Older job ads convert better.** Target companies still advertising after 5+ days — urgency rises with time.
